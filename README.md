# Vérificateur de mot de passe sécurisé
## Projet pour portfolio cybersécurité débutant python

C'est un programme qui permet de verifier la sécurité d'un mot de passe.
Il test si le MDP à au moins 8 caractères, une Majuscule et un chiffre, puis calcule un score de sécurité.
C'est mon tout premier code en python et qui me permet d'apprendre et progresser dans le but de me former en cybersécruité. 

### Fonctionnalités
- Vérifie la longueur du mot de passe
- Vérifie la présence d'au moins une majuscule
- Vérifie la présence d'au moins un chiffre
- Calcule un score de sécurité 
- Affiche si le mot de passe est accepté ou refusé 
- Créer un fichier automatiquement et enregistre les mots de passe en clair 
- Créer une boucle temps que le mots de passe ne correspond pas aux critères 
 
 ### Comment l'utiliser 
 1. Clonner le dépôt GitHub ou télécharger le fichier.
 2. Ouvrir le fichier 'psdw.py' dans VS Code.
 3. Exécuter le programme.
 4. Saisir un mot de passe pour tester sa sécurité. 

 ### Technologies 
 - Python 3
 - VS code
